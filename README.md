

# My-Raystation-repo
Sharing raystation scripts for automation in the clinic. 

## Folders in this repo:

**DLS Contour Evaluation form:**

Below you see an screenshot of the gui that pops-up in ray station in the upper right corner. 
Also, you see an example of the json file where the filled in data is written to: example_prostaat_loge_dls-f3d35e92-3300-4573-9e69-d960599d6064.json. Contours are exported in a numpy array zip file (.npz) in the RayStation format voxelroi. These exported contours can be imported again to RayStation for offline processing. The idea is to calculate similarity coefficients and Added Path Length.
